
# Dashboard Parqueo Vehicular de Guatemala — Visión Ejecutiva 2026

Este proyecto presenta un análisis completo del parque vehicular de Guatemala, utilizando visualizaciones ejecutivas para comprender la composición, antigüedad, uso y distribución del parque automotor a nivel nacional.

El objetivo principal es ofrecer una vista clara y accionable para la toma de decisiones estratégicas en sectores como transporte, movilidad urbana, seguridad vial y planificación gubernamental.

---

## 📊 **Principales Indicadores**

- **6M** — Total del parque vehicular
- **67%** — Vehículos recientes (modelos modernos)
- **95%** — Uso personal
- **5%** — Uso de trabajo
- **7 años** — Edad promedio del parque
